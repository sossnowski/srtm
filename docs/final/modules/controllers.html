mainController¶

mainController.getUavosFromRedis()¶

Get uavos from redis and filter them by source type. Return list of filtered uavos.

Returns

Array – Filtered uavos from redis

mainController.mergeUavos(uavos)¶

Merge uavos if it came from different sources. If it exists in cache, it is already merged.

Arguments

  • uavos (Object) – Object consisting all uavos

Returns

Object – Composed with mergedUavos, uavosToAdd and uavosToUpadte
