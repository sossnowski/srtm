Elevation¶

Controller¶

point(coordinates)¶

Gets specific point and elevation for this point.

Arguments

  • coordinates (Tuple) – Coordinates of a point.

Returns

Object – Consist of specific point and evaluated elevation.

area(areasData, distance)¶

Gets grid points of specific area and evaluate elevation for every part of this area.

Arguments

  • areasData (Object) – Object containing data about area.

  • distance (Number) – Segments length

Returns

Object – Contains updated with new features areasData object.

Routes¶

Provides endpoints used in evaluation of an elevation. We can distinguish two specific endpoints: ‘/point/:coordinates’ and ‘/area/:distanceBetweenPoints’. Choosing an endpoint results in invoking functions from elevationController.js (point() or area()).

Service¶

pointFromCoordinates(coordinates)¶

Translate coordinates presented in text to an object of lon and lat values.

Arguments

  • coordinates (String) – Lat and Lon values in text

Returns

Object – {Lon, Lat} values

getElevation(requestPoint)¶

Get elevation for selected point.

Arguments

  • requestPoint (Object) – Contains coordinates in {lat, lon} form.

Returns

Number – Elevation height.

getPointsGridOfArea(area, pointDistance)¶

Evaluate mesh points of given area.

Arguments

  • area (Object) – Feature object.

  • pointDistance (Number) – Length of a segment

Returns

Object – Contains every point coordinates.

getElevationsFromPointsGrid(points)¶

Evaluate elevation using mesh points returned by getPointsGridOfArea function.

Arguments

  • points (Object) – Collection of mesh points.

Returns

Array – Elevation for every given point.

getStatiticsDataFromElevationsArray(elevationsArray)¶

Gets max, min values of elevation and average elevation.

Arguments

  • elevationsArray (Array) – Array of elevation measurements.

Returns

Object – Contains min, max values and average elevation of elevationsArray.

splitPolygon(polygonData, polygonGeojson, height)¶

Splits given polygon by limiting single part height.

Arguments

  • polygonData (Object) – Polygon feature.

  • polygonGeojson (Object) – Geojson of polygon.

  • height (Number) – Max height of part of splitted area.

Returns

Array – Splited polygon array with properties(minH, maxH, avg, amslH).
